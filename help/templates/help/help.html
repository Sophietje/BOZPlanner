{% trans 'Scheduling a meeting' %}

{% blocktrans %}

In order to schedule a meeting, you should navigate in the menu to "Meetings" and click on the add button on top of the page. You should then fill in the location of the meeting (e.g. RA-5148), at what time the meeting starts (in the following format: dd/mm/yyyy hh:mm), the datetimepicker can help to fill in this field, then you should fill in at what time the meeting will end (This work exactly the same as the previous field). Next, the organization of the OLC meeting should be filled in (e.g. BMS). Finally click on the save button to save the meeting.

{% trans 'Users information' %}

{% blocktrans %}

You can find all users of the system in the menu named "Users". This menu serves multiple purposes, firstly you can thus find all users of the system, thus students, BOZ employees and persons associated to an OLC. Secondly, an account can be created by means of the add button (found in the header of the page). Thirdly, data about the users' data can be changed when requested or even a user can be deleted from the list.

How do I add a user?

In order to add a user, you should as previously mentioned click on the add button on top of the "Users" page. Then you should fill in the name, email and to which organization the user is associated to. Afterwards, the person can be added by submitting the form. The person for whom you filled in the data can now access the system.

How do I change the data of a user?

The data of a user can be changed by clicking on the change button in the list associated to the user. Then, the data of the user pops up enabling you to make changes to it. Afterwards, the data should be submitted.

How do I delete a user?

A user can be deleted by clicking on the delete button in the list associated to the user. Then, you will be asked whether you are certain of deleting the user. Afterwards, click on the deactivate button to permanently deactivate the user.

{% trans 'Minutes' %}

{% blocktrans %}

In order to download minutes from meetings that took place in the past, you should navigate to the menu named "Minutes". In this menu, the past meetings are listed. When minutes are uploaded by the student, a download button can be found on the right hand side in the list. It could be the case that a student has uploaded multiple versions of the minutes. In that case, you are able to chose which version you would like to download.
